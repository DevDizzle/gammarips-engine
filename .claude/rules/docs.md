# Rules for docs/

- `docs/TRADING-STRATEGY.md` is the canonical execution policy. Update it whenever execution policy changes.
- `docs/DECISIONS/` contains the decision trail. Add a new dated file for every policy change.
- `docs/research_reports/_archive/` is historical — do not treat as authoritative.
- `docs/research_reports/INTELLIGENCE_BRIEF.md` is the live research state — update it when hypotheses are tested or resolved.
- Prefer updating existing docs over creating new ones.
